Detailed Description

Class for managing communication with Redis.

This class manages all protocol related stuff, i.e communicating with redis, you should never need to directly access this class See http://redis.io/topics/protocol for specification details

_readBulk ( ) [private]

Read a bulk reply from redis.

Reads a bulk reply, the number of responses is the first line, if it's -1 then null is returned which means the request doesn't exist
If the length is 0 then an empty string is returned
Else $length bytes is read and returned.
Lastly if debugging is enabled then the EOL read is noted and the EOL marker is read so it's stripped from the response.

Returns:
string|null the response

connect ( )

connects to redis if not already connected

Connects to redis if not already connected, sets the stream timeout to the value of STREAM_TIMEOUT in Redis::$_config which defaults to one hour, if a client is running a blocking operation i.e. brpop for more than an hour it will disconnect. It will attempt to pull configuration information from Redis::$_config if a UNIX_SOCKET is set then that will be preferred over a TCP connection

Returns:
$this
Exceptions:
RedisExceptionif connection fails

response ( $  raw = false)

Gets the response from redis.

Reads a response from redis, if $raw is false then format() is called on the response if it's a single line response.
If the response is a bulk or multi bulk then Protocol::_readBulk() and Protocol::_readMultiBulk() are called respectively
